## Notação Big-O
===
- Uma notação especial para determinar o desempenho de um algoritmo
- Não leva em consideração SO, poder de processamento (hardware)

### Vetores não ordenados
- Inserção: Big-O Constante - O(1)
- Pesquisa: Big-O Linear - O(n)
- Exclusão: Big-O -> O(2n) - O(n)
- Vertor permite itens duplicados
  - Inserção O(1); pesquisa e exclusão O(n)
- Vetor não permite itens duplicados
  - Inserção O(n); pesquisa e exclusão O(n)
